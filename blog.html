#1 HTML : The backbone of a Website

Author: Rahul Ahmed Sezan| Date: 06/04/2022| Tags:html, css, web design, web development

Html stands for Hyper Text Markup Language. Everything we see in a website is an html element. Nowaday many advanced technology being used to developing website but every technology converts their elements into html elements. It's a text document saved with the extension .html or .htm that contains texts and some tags written between "<>" which give the instructions needed to configure the web page.
After completing this assingnment I have learned a lot of things about HTML and CSS. Some of highling things are:

  • There are a lot of tags in html but some of these tags are called semantic tags. Semantic tags explain what the element is or what is the role of the element in a web page. Semantic elements are the features of HTML5.

  • For aligning elements in a web page vertically of horizontally Flexbox is the best. Flexbox is a powerful tool of css. With flexbox it's very easy to align elements in both direction.

  • There are many convention of writing html code. We can validate our code with w3 validator. Writing code in a standard way is multiway benificial for a developer because it's easy to maintain code and find out bugs or errors. Doing this assignment I have learned many standard way of writing html code. For example writing < nav> instead of < div class="nav"> is more convenient.
